I'm Margo K. Kelly



Welcome to my resume site. Below you will find a link to my resume and a link to a diagram of this website. The site contains my implementation of Forrest Brazeal's Cloud Resume Challenge. I enjoyed this challenge tremendously. The visitor counter at the foot of the resume page is implemented using AWS Lambda, DynamoDB and API Gateway. A Javascript API call in the browser triggers the AWS Lambda function. AWS Lambda, API Gateway and DynamoDB were all deployed using AWS SAM. GitHub Actions is used to automatically update the site's S3 contents when new code is pushed to the repository. I am currently working on the backend CI/CD and TDD portions of the challenge.

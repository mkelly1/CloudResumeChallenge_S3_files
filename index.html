Welcome to my resume website. In the navbar you will find links to my resume and a link to a diagram of the website. This site contains my implementation of Forrest Brazeal's Cloud Resume Challenge. The visitor counter at the foot of my resume page is incremented by an AWS Lambda function which increments and stores the count in DynamoDB. A Javascript API call in the browser triggers the AWS Lambda function. AWS Lambda, API Gateway and DynamoDB were all deployed using AWS SAM.

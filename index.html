I'm Margo K. Kelly



Welcome to my resume site. Below you will find links to my resume and a diagram of this website. This site contains my implementation of Forrest Brazeal's Cloud Resume Challenge. I enjoyed this challenge tremendously. The resume page implements an unassuming visitor counter in the footer. One had to use AWS API Gateway, Lambda, DynamoDB. All had to be configured and deployed using AWS SAM. Also GitHub Actions had to be used to automatically update the site's S3 contents when new code is pushed to the repository. I am currently working on the backend CI/CD and TDD portions. For further challenge details, please see the Cloud Resume Challenge page and please take a look at the site diagram of this site.
